# Disclaimer
 
## Educational Use Only
 
This project was created solely for academic and educational purposes as part of a college cybersecurity project. It is intended to demonstrate how spyware attack chains work at a conceptual level, to help students and readers understand threats so they can better defend against them.
 
## What Has Been Removed
 
To prevent misuse, the following components have been deliberately removed or redacted from this repository:
 
- **C2 server endpoint** — the IP/host the client connects to has been redacted
- **VM detection logic** — removed entirely
- **Persistence mechanism** — removed entirely
- **Process concealment code** — removed entirely
- **Compiled binary** — the `.exe` is not included and will not be provided
The code in this repository **cannot be compiled into a functional spyware tool** as-is.

The authors of this project accept no responsibility for any misuse of the concepts or partial code demonstrated here. This repository is provided for educational reference only.
 
Deploying spyware — even modified versions of this code — against systems you do not own and have not been given explicit permission to test is **illegal** in most jurisdictions, including under laws such as:
 
- The Computer Fraud and Abuse Act (CFAA) — USA
- The Computer Misuse Act — UK
